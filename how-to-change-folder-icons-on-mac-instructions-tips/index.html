How to Change Folder Icons on Mac: Instructions + Tips

20 March 2026

Customizing folder icons on your Mac is a simple way to make your workspace feel more organized—and a lot more personal. Here’s how to do it, plus a few tips to make it look clean and intentional.

Instructions

1. Choose your icon

Find or create an image (PNG works best, ideally with a transparent background).

2. Copy the image

Open it in Preview and press Command (⌘) + C.

3. Open folder info

Right-click the folder → click “Get Info.”

4. Select the folder icon

In the top-left corner of the info window, click the small folder icon until it’s highlighted.

5. Paste your new icon

Press Command (⌘) + V and it’ll instantly update.

Tips to Make It Look Good

  • Stay consistent: Use icons from the same style or set so everything feels cohesive

  • Use colour coding: Assign colours for categories (e.g., blue = school, green = finance)

  • Keep it simple: Overly detailed icons can look messy at small sizes

  • Try emojis: Quick, easy, and surprisingly effective (📂✨📸)

  • Save a mini icon pack: Makes it faster to reuse and stay consistent
